# Introducción a la física computacional

El principal objetivo del curso es la introducción al uso de los ordenadores y la simulación computacional como herramientas básicas de la física.

Utilizaremos Python debido a su flexibilidad. Usaremos Spyder y Jupyter, ambos están disponibles en el entorno Anaconda que podéis descargar aquí:
https://www.anaconda.com/download

Además de Python, se mostrará el uso de otras herramientas útiles como el acceso y uso de servidores remotos (Linux). Para esto, si trabajamos en Windows se recomienda la instalación de Moba: 
https://mobaxterm.mobatek.net/

Este repositorio incluye: 
1. Cuadernos de introducción y práctica a Python
2. Presentación de tres problemas de física computacional:
   1. Modelo de Ising
   2. Modelo de Hopfield
   3. Partícula cuántica en un pozo de potencial (ecuación de Schrödinger)
